OTP Verifcation system

##Project Overview

This project is a simple OTP ( One-Time-Password) authentication system developed using python the system generates a random 4-digit otp and verifies user input against the generate otp

##Features

*Random OTP generation using python
*user input handling
*Conditional verification logic
*interactive command-line interface

##Technologies Used

*python
*Random Module
*Conditional statements

##How it works

*Generates a random 4-digit OTP
*Prompts the user to enter the OTP
*validates the input against the generated otp
*Displays sucess or failure message
