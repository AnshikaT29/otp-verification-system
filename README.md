## 🔐 OTP Verification System

### 📌 Project Overview
This project is a simple OTP (One-Time Password) authentication system developed using Python. The system generates a random 4-digit OTP and verifies user input against the generated OTP.

### ✨ Features
✔ Random OTP generation using Python  
✔ User input handling  
✔ Conditional verification logic  
✔ Interactive command-line interface  

### 🛠 Technologies Used
• Python  
• Random Module  
• Conditional Statements  

### ⚙ How It Works
➡ Generates a random 4-digit OTP  
➡ Prompts the user to enter the OTP  
➡ Validates the input against the generated OTP  
➡ Displays success or failure message
